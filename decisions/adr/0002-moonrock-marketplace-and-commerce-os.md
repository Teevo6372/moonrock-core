# ADR-0002: Separate Moonrock Marketplace from Moonrock Commerce OS

- Status: Accepted
- Date: 2026-07-22
- Decision owner: Stephen Tyler Jr.

## Context

The ecosystem needs both a customer-facing commerce identity and an internal operating product. The former name `Project Midnight` did not align with the Moonrock naming system.

## Decision

Use **Moonrock Marketplace** for the commerce business and customer-facing marketplace.

Use **Moonrock Commerce OS** for the internal AI-assisted commerce operating system that supports sourcing, evaluation, pricing, listing, fulfillment, inventory, and analytics.

## Consequences

- Project Midnight is retired as an active product name.
- Marketplace documentation describes channels, offers, customers, partners, and catalog strategy.
- Commerce OS documentation describes software, data, calculations, integrations, controls, and automation.
- Commerce OS may support Moonrock Marketplace and external sales channels without becoming the customer-facing marketplace brand.
