# ADR-0001: Use Moonrock Enterprises as the Umbrella

- Status: Accepted
- Date: 2026-07-22
- Decision owner: Stephen Tyler Jr.

## Context

The ecosystem requires an umbrella identity broad enough to support operating businesses, products, platforms, partnerships, property, and future expansion. `Moonrock Holdings` implies a primarily asset-holding structure and does not fully describe the active operating ecosystem.

## Decision

Use **Moonrock Enterprises** as the umbrella name for repository architecture and ecosystem planning.

The current operating units are:

- Moonrock Marketing
- Moonrock Systems
- Moonrock Marketplace
- Moonrock Biophilic
- Moonrock Properties

Moonrock Commerce OS is a shared product and platform capability, not a replacement for Moonrock Marketplace.

## Consequences

- New architecture documents use Moonrock Enterprises.
- Existing legal entity names are not automatically changed by this repository decision.
- Legal, tax, banking, contract, and registration changes require separate review and execution.
- Historical references may remain where they accurately describe prior decisions.
